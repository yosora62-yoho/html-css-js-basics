📄 HTML - โครงสร้างเว็บไซต์

HTML คืออะไร? HTML ย่อมาจาก HyperText Markup Language คือภาษาที่ใช้สำหรับสร้างโครงสร้างของเว็บไซต์ เหมือนกับโครงกระดูกของมนุษย์ ถ้าไม่มี HTML เว็บไซต์ก็จะไม่มีอะไรเลย

ใช้ทำอะไร? ใช้กำหนดว่าบนเว็บไซต์จะมีอะไรบ้าง เช่น หัวข้อ, ย่อหน้า, รูปภาพ, ปุ่ม, ฟอร์มรับข้อมูล, ตาราง, ลิงก์ต่างๆ

ใช้ยังไง? เขียนแท็ก <> กำกับเนื้อหา เช่น <h1>หัวข้อ</h1> browser จะอ่านแล้วแสดงผลเป็นหัวข้อใหญ่

🏷️ แท็ก HTML ที่ต้องรู้จัก

📌 แท็กพื้นฐาน

<!-- นี่คือคอมเมนต์ ไม่แสดงผล -->

<h1>หัวข้อระดับ 1</h1>
<h2>หัวข้อระดับ 2</h2>
<h3>หัวข้อระดับ 3</h3>

<p>นี่คือย่อหน้า ใช้สำหรับเขียนข้อความยาวๆ</p>

<div>กล่องสำหรับจัดกลุ่มเนื้อหา</div>

<span>ใช้สำหรับตกแต่งข้อความเฉพาะส่วน</span>
🔍 อธิบาย:
• <h1>~<h6> = หัวข้อ ตัวเลขยิ่งน้อยตัวยิ่งใหญ่ (h1 ใหญ่สุด h6 เล็กสุด)
• <p> = ย่อหน้า ขึ้นบรรทัดใหม่ มีระยะห่างด้านบนล่าง
• <div> = กล่องแบบ block กว้างเต็มแนวนอน ใช้จัดโครงสร้าง
• <span> = กล่องแบบ inline กว้างเท่าเนื้อหา ใช้ตกแต่งเฉพาะส่วน
• <!-- --> = คอมเมนต์ ไว้จดโน้ม ไม่แสดงผลบนหน้าเว็บ
🎯 ลองดูผลลัพธ์:

หัวข้อระดับ 2

นี่คือตัวอย่างย่อหน้า ข้อความจะแสดงแบบนี้

กล่อง div มีพื้นหลังสีเทา
span ทำให้ข้อความนี้เป็นสีแดง

</a>

<!-- ลิงก์ไปหน้าอื่นในเว็บเรา -->

<img src="https://picsum.photos/200" 
     alt="รูปตัวอย่าง"
     width="200">
🔍 อธิบาย:
• <a> = anchor สร้างลิงก์ href = ปลายทางที่จะไป
• target="_blank" = เปิดลิงก์ในแท็บใหม่
• <img> = รูปภาพ src = ที่อยู่รูป, alt = ข้อความเมื่อโหลดรูปไม่ได้ (สำคัญต่อ SEO)
• width/height = กำหนดขนาดรูป (พิกเซล)

🎨 CSS - การตกแต่งและจัดวาง

CSS คืออะไร? CSS ย่อมาจาก Cascading Style Sheets คือภาษาที่ใช้สำหรับตกแต่งเว็บไซต์ให้สวยงาม เหมือนกับเสื้อผ้าและเครื่องประดับของมนุษย์

ใช้ทำอะไร? กำหนดสี ขนาดตัวอักษร พื้นหลัง ระยะห่าง ตำแหน่ง การจัดวาง อนิเมชั่น ทำให้เว็บน่าดูและใช้งานง่าย

ใช้ยังไง? เขียน selector แล้วกำหนด property: value เช่น h1 { color: red; font-size: 30px; }

🎨 พื้นฐาน CSS ที่ต้องรู้

🔍 Selector (ตัวเลือก)

/* Element Selector - เลือกตามชื่อแท็ก */
p {
  color: blue;
}

/* Class Selector - ใช้ . นำหน้า */
.box {
  background: yellow;
  padding: 10px;
}

/* ID Selector - ใช้ # นำหน้า (ใช้แค่ 1 อัน) */
#header {
  background: black;
  color: white;
}

/* Multiple Selector - เลือกหลายอัน */
h1, h2, h3 {
  font-family: 'Prompt';
}
🔍 อธิบาย:
• Element Selector = เลือกตามชื่อแท็ก เช่น p, div, h1 (แต่งทุกอันที่เหมือนกัน)
• Class Selector = ใช้ .(จุด) ตามด้วยชื่อ class ใช้ซ้ำได้หลายอัน
• ID Selector = ใช้ # ตามด้วยชื่อ id ใช้ได้แค่ 1 อันต่อหน้า
• ความสำคัญ: ID > Class > Element

📦 Box Model (สำคัญมาก!)

.box {
  width: 200px;
  padding: 20px;     /* ขอบด้านใน */
  border: 2px solid black; /* ขอบ */
  margin: 30px;      /* ขอบด้านนอก */
  box-sizing: border-box;
}
🔍 Box Model อธิบาย:
ทุก element ใน HTML เป็นกล่อง 4 ชั้น:
• Content = เนื้อหาข้างใน (ข้อความ, รูป)
• Padding = ระยะห่างระหว่าง content กับ border (ขยายกล่องออก)
• Border = เส้นขอบ
• Margin = ระยะห่างจาก element อื่น
• box-sizing: border-box; ทำให้ width รวม padding+border (ง่ายกว่า)
🎯 ดูตัวอย่าง Box Model:
เนื้อหา
padding:20px
border:3px
margin:20px

🔲 Flexbox (การจัดวาง)

.container {
  display: flex;
  flex-direction: row;      /* row, column */
  justify-content: center;  /* แนวหลัก */
  align-items: center;      /* แนวรอง */
  gap: 20px;               /* ระยะห่าง */
  flex-wrap: wrap;         /* ขึ้นบรรทัดใหม่ */
}
🔍 Flexbox อธิบาย:
• display: flex; = เปิดโหมด Flexbox
• flex-direction = ทิศทาง (row=แนวนอน, column=แนวตั้ง)
• justify-content = จัดแนวแกนหลัก (center, space-between, flex-start, flex-end)
• align-items = จัดแนวแกนรอง (center, stretch)
• gap = ระยะห่างระหว่างลูก
• flex-wrap = ให้กระโดดบรรทัดเมื่อที่ว่างไม่พอ
🎯 ดูตัวอย่าง Flexbox:
ชิ้นที่1
ชิ้นที่2
ชิ้นที่3

↑ จัดแนวนอน กึ่งกลาง ระยะห่างเท่ากัน

🎨 สีและพื้นหลัง

/* สีตัวอักษร */
color: red;
color: #FF0000;      /* HEX */
color: rgb(255,0,0); /* RGB */

/* พื้นหลัง */
background-color: #f0f0f0;
background-image: url('bg.jpg');
background: linear-gradient(135deg, #667eea, #764ba2);
🔍 อธิบาย:
• สีแบบชื่อ = red, blue, green, yellow
• สีแบบ HEX = #RRGGBB (00-FF) เช่น #FF0000 = แดง
• สีแบบ RGB = rgb(แดง, เขียว, น้ำเงิน) 0-255
• Gradient = การไล่สี สวยงาม
• rgba() = RGB + alpha (ความโปร่งใส) 0-1

✏️ ตัวอักษร

p {
  font-family: 'Prompt', sans-serif;
  font-size: 16px;
  font-weight: bold;
  text-align: center;
  line-height: 1.5;
  text-decoration: underline;
}
🔍 อธิบาย:
• font-family = ฟอนต์ที่ใช้ (เรียงจากตัวแรกที่มี)
• font-size = ขนาดตัวอักษร (px, rem, em)
• font-weight = ความหนา (normal, bold, 100-900)
• text-align = จัดแนว (left, center, right, justify)
• line-height = ระยะห่างระหว่างบรรทัด
• text-decoration = ขีดเส้น (underline, line-through, none)

📱 Responsive (จอเล็ก-ใหญ่)

/* จอใหญ่ (คอมพิวเตอร์) */
.container {
  width: 1200px;
  margin: auto;
}

/* จอเล็ก (มือถือ) */
@media (max-width: 768px) {
  .container {
    width: 100%;
    padding: 10px;
  }
  .menu {
    flex-direction: column;
  }
}
🔍 อธิบาย:
• Media Query = เปลี่ยน CSS ตามขนาดหน้าจอ
• @media (max-width: 768px) = เมื่อจอกว้างน้อยกว่า 768px
• ทำให้เว็บดูดีทั้งมือถือและคอมพิวเตอร์
• ใช้หน่วย vw/vh = viewport width/height
• ใช้ % แทน px เพื่อความยืดหยุ่น
🎯 ลองปรับขนาดหน้าจอ:
จอกว้างมากกว่า 768px

⚡ JavaScript - ทำให้เว็บมีชีวิต

JavaScript คืออะไร? JavaScript คือภาษาโปรแกรมที่ทำงานบนเบราว์เซอร์ ทำให้เว็บไซต์มีชีวิต โต้ตอบกับผู้ใช้ได้ เหมือนสมองและระบบประสาทของมนุษย์

ใช้ทำอะไร? กดปุ่มแล้วมีอะไรเกิดขึ้น, นับเลข, ตรวจสอบฟอร์ม, โหลดข้อมูล, สร้างเกม, ทำอนิเมชั่น, เชื่อมต่อ API

ใช้ยังไง? เขียนโค้ดใน <script> หรือไฟล์ .js แล้ว browser จะรันให้

⚡ พื้นฐาน JavaScript ที่ต้องรู้

📦 ตัวแปร (Variables)

// let - เปลี่ยนค่าได้ (ใช้บ่อยสุด)
let name = "นินิ";
name = "นินินานา";  // เปลี่ยนได้

// const - ค่าคงที่ เปลี่ยนไม่ได้
const PI = 3.14159;
// PI = 3;  // ERROR!

// ประเภทข้อมูล
let text = "ข้อความ";     // String
let number = 10;         // Number
let isFun = true;        // Boolean
let fruits = ["แอปเปิ้ล", "กล้วย"]; // Array
let user = {             // Object
  name: "นินิ",
  age: 5
};
🔍 อธิบาย:
• let = ประกาศตัวแปร เปลี่ยนค่าทีหลังได้ (ใช้บ่อยที่สุด)
• const = ค่าคงที่ เปลี่ยนไม่ได้ (ใช้กับค่าที่ไม่เปลี่ยน)
• String = ข้อความ ใส่ "" หรือ ''
• Number = ตัวเลข ทั้งจำนวนเต็มและทศนิยม
• Boolean = จริง/เท็จ (true/false)
• Array = รายการข้อมูล ใช้ []
• Object = ข้อมูลแบบมีชื่อ ใช้ {}
🎯 ลองเล่นกับตัวแปร:

🔄 เงื่อนไข (if/else)

let score = 75;

if (score >= 80) {
  console.log("เกรด A");
} else if (score >= 70) {
  console.log("เกรด B");
} else if (score >= 60) {
  console.log("เกรด C");
} else {
  console.log("เกรด F");
}
🔍 อธิบาย:
• if (เงื่อนไข) { ... } = ถ้าเงื่อนไขเป็นจริง ทำใน {}
• else if = ถ้าไม่ตรง if แต่ตรงเงื่อนไขนี้
• else = ถ้าไม่ตรงเงื่อนไขไหนเลย
• === = เท่ากัน (เปรียบเทียบทั้งค่าและชนิด), !== = ไม่เท่ากัน
• >, <, >=, <= = มากกว่า, น้อยกว่า, มากกว่าหรือเท่ากัน, น้อยกว่าหรือเท่ากัน
• && = และ, || = หรือ, ! = ไม่
🎯 ทดสอบเงื่อนไข:

🔄 การวนซ้ำ (Loop)

// for loop - รู้จำนวนรอบ
for (let i = 0; i < 5; i++) {
  console.log("รอบที่ " + i);
}

// while loop - ไม่รู้จำนวนรอบ
let count = 0;
while (count < 5) {
  console.log("นับ: " + count);
  count++;
}
🔍 อธิบาย:
• for (เริ่มต้น; เงื่อนไข; เพิ่มค่า) = ใช้เมื่อรู้จำนวนรอบ
• เริ่มต้นที่ 0, วนจนกว่าเงื่อนไขเป็นเท็จ, เพิ่มค่าทีละ 1
• while (เงื่อนไข) = ใช้เมื่อไม่รู้จำนวนรอบ วนตราบใดที่เงื่อนไขเป็นจริง
• forEach = วน array แบบง่าย
🎯 ลองวนซ้ำ:

⚙️ ฟังก์ชัน (Function)

// ประกาศฟังก์ชัน
function sayHello(name) {
  return "สวัสดี " + name;
}

// เรียกใช้ฟังก์ชัน
let message = sayHello("นินิ");
console.log(message); // "สวัสดี นินิ"

// Arrow function (แบบสั้น)
const add = (a, b) => a + b;
🔍 อธิบาย:
• function ชื่อ(parameter) { ... } = สร้างฟังก์ชัน
• return = คืนค่ากลับ (ถ้าไม่มี return จะได้ undefined)
• parameter = ตัวแปรที่รับค่าตอนเรียกใช้
• Arrow function = วิธีเขียนแบบสั้น (=>)
• ใช้ฟังก์ชันเพื่อไม่ต้องเขียนโค้ดซ้ำ
🎯 ลองใช้ฟังก์ชัน:

🎯 การเข้าถึงและเปลี่ยน HTML

// หา element
let title = document.getElementById("myTitle");
let box = document.querySelector(".box");

// เปลี่ยนข้อความ
title.innerText = "ข้อความใหม่";

// เปลี่ยน HTML ข้างใน
box.innerHTML = "<strong>ตัวหนา</strong>";

// เปลี่ยน CSS
title.style.color = "red";
title.classList.add("highlight");
🔍 อธิบาย:
• document.getElementById("id") = หา element จาก id
• document.querySelector(".class") = หา element แรกที่ตรง selector
• .innerText = เปลี่ยนข้อความข้างใน (HTML ไม่ถูก render)
• .innerHTML = เปลี่ยน HTML ข้างใน (render แท็กได้)
• .style.ชื่อProperty = เปลี่ยน CSS แบบ inline
• .classList.add/remove/toggle = จัดการ class

🖱️ Event (การตอบสนอง)

// คลิก
button.onclick = function() {
  alert("ถูกคลิก!");
};

// addEventListener (แนะนำ)
button.addEventListener("click", () => {
  console.log("คลิก");
});

// Event ประเภทต่างๆ
// onclick, ondblclick, onmouseenter
// onkeydown, onkeyup
// onsubmit, onchange
🔍 อธิบาย:
• onclick = เมื่อคลิก
• ondblclick = ดับเบิลคลิก

📖 โค้ด JavaScript ของเกมนี้

// เกมทายตัวเลข - เรียนรู้ JavaScript จากการเล่น!
let secretNumber;      // ตัวเลขลับ
let attempts;          // จำนวนครั้งที่ทาย
let highScore;         // สถิติสูงสุด

// ฟังก์ชันเริ่มเกมใหม่
function initGame() {
    secretNumber = Math.floor(Math.random() * 100) + 1;
    attempts = 0;
    updateDisplay();
}

// ฟังก์ชันตรวจสอบการทาย
function checkGuess() {
    let guess = Number(input.value);
    attempts++;
    
    if (guess === secretNumber) {
        // ถูกต้อง!
        messageDiv.innerText = "🎉 ถูกต้อง! เก่งมาก!";
        if (highScore === null || attempts < highScore) {
            highScore = attempts;
        }
    } else if (guess > secretNumber) {
        // มากไป
        messageDiv.innerText = "📈 มากไป! ลองน้อยกว่านี้";
    } else {
        // น้อยไป
        messageDiv.innerText = "📉 น้อยไป! ลองมากกว่านี้";
    }
}
🔍 อธิบายโค้ดเกม:
• Math.random() = สุ่มตัวเลข 0-1 (ไม่รวม 1)
• Math.floor() = ปัดเศษลง ทำให้ได้เลขจำนวนเต็ม
• if/else = ตรวจสอบเงื่อนไขว่าทายถูกหรือผิด
• localStorage = เก็บสถิติสูงสุดไว้ แม้ปิด browser
• document.getElementById() = เข้าถึง element ใน HTML
• .innerText = เปลี่ยนข้อความใน element
